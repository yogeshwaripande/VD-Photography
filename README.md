## 📖 About the Project

VD-Photography is a professional portfolio and business management web application designed for photographers.
It allows users to explore portfolios and events, while admins can manage orders, billing, and business operations efficiently.
## 🚀 Features

### 👤 User Module
* View Home, Portfolio & Events
* Submit Enquiry
* Place Order
* View Order Status
* View Bill & Download Photos

### 👨‍💼 Admin Module
* Manage Portfolio & Events
* Handle Orders & Bookings
* Generate Bill & Quotation
* Manage Expenses
* Auto-delete images (7/14 days)
* 
## 🛠️ Tech Stack

* 💻 Frontend: HTML, CSS, JavaScript
* ⚙️ Backend: Node.js, Express.js
* 🗄️ Database: MongoDB
* 
## 🔐 Security Features

* QR Code-based secure photo access
* Data validation and authentication
